Credential
==========

Cette application doit pouvoir fournir des méthodes permettant de :

 - Contrôler les accès
 - Gérer les sessions
 - Gérer les comptes
 - Offrir des moyens de confidentialités, d'intégrités et de disponibilités des données
 - Autorisés et gérer les clients de cette application
 - Contrôler les entrées
 - Gérer les exceptions
 - Protéger contre le rejeu
 - Auditer
